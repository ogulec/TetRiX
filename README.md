# TetRiX

**TetRiX** is a Retro Tetris game powered by **Flutter**.

It's a self Flutter learning fun-app which is re-designed and re-coded.

*Repo forked from* [boyan01/flutter-tetris](https://github.com/boyan01/flutter-tetris)
